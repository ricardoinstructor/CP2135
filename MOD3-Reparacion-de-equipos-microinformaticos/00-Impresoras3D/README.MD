# 📦 Recopilación de Sitios con Modelos Gratuitos para Impresión 3D

A continuación, se presenta una lista de sitios web donde puedes encontrar modelos 3D listos para imprimir, totalmente gratuitos (algunos bajo licencias específicas, por favor revisa los términos de uso en cada plataforma).

---

## 🔗 Sitios Web

- **[Thingiverse](https://www.thingiverse.com/)**  
  Uno de los repositorios más grandes y populares. Comunidad activa y amplia variedad de diseños.

- **[Printables](https://www.printables.com/)**  
  Plataforma moderna respaldada por Prusa. Alta calidad, retos comunitarios y muchos modelos gratuitos.

- **[Cults3D (Free section)](https://cults3d.com/en/3d-models/free)**  
  Contiene tanto modelos gratuitos como de pago. Filtra por "Free" para acceder a los gratuitos.

- **[MyMiniFactory](https://www.myminifactory.com/)**  
  Modelos verificados y testeados en impresora. Cuenta con una sección gratuita muy completa.

- **[Thangs](https://thangs.com/)**  
  Buscador de modelos 3D que agrega resultados de múltiples fuentes. Incluye muchos gratuitos.

- **[GrabCAD](https://grabcad.com/library)**  
  Enfocado en ingeniería y diseño técnico. Gran cantidad de modelos industriales y mecánicos.

- **[Pinshape (Free models)](https://pinshape.com/)**  
  Comunidad de diseñadores. Usa el filtro “Free” para encontrar modelos sin costo.

- **[Yeggi](https://www.yeggi.com/)**  
  Motor de búsqueda especializado en modelos 3D para impresión. Muestra resultados de varias plataformas.

- **[NASA 3D Resources](https://nasa3d.arc.nasa.gov/models)**  
  Modelos 3D oficiales de naves, planetas, rovers y más, proporcionados por la NASA. Dominio público.

- **[NIH 3D Print Exchange](https://3dprint.nih.gov/)**  
  Modelos biomédicos y científicos, muchos de ellos listos para impresión y de uso educativo.

---

## ⚠️ Notas Importantes

- Siempre revisa la **licencia** del modelo antes de usarlo (especialmente si es para uso comercial).
- Algunos sitios requieren crear una cuenta gratuita para descargar.
- Los formatos más comunes son `.STL`, `.OBJ` y `.3MF`.
